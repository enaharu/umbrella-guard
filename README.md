# 傘おかんアプリ

位置情報を利用して、一定時間同じ場所に滞在したあとに移動を開始したタイミングで「傘忘れてない？」とローカル通知する Expo + React Native + TypeScript のMVPです。

## 機能

- Foreground / Background Location 権限の取得
- 半径100m以内で10分以上の滞在検知
- 滞在地点から300m以上移動した時の移動開始検知
- ローカル通知
- 通知後60分のクールダウン
- ホーム、設定、ログ画面
- 記事検証用のデバッグ表示

## 起動

```bash
npm install
npm run start
```

バックグラウンド位置情報と通知の検証は、Expo Go では制限される場合があります。実機検証では development build の利用を推奨します。

## 検証メモ

設定画面で滞在時間、移動距離、クールダウン時間、デバッグモードを変更できます。ホーム画面の「通知テスト」でローカル通知の表示を確認できます。
