傘おかん プライバシーポリシー

本プライバシーポリシーは、iOSアプリ「傘おかん」における利用者情報の取り扱いについて説明するものです。

制定日: 2026年7月14日

1. 利用する情報

傘おかんは、滞在場所から離れたタイミングで忘れ物確認の通知を行うため、利用者の端末上で位置情報を使用します。 また、通知機能を利用して、利用者にリマインダーを表示します。

2. 位置情報の取り扱い

位置情報は、滞在場所の検知および忘れ物確認通知のために端末内で使用します。 位置情報を外部サーバーへ送信することはありません。

3. 端末内に保存される情報

アプリの設定、通知の状態、検知に必要な一時的な位置情報、動作ログは、利用者の端末内に保存される場合があります。 これらの情報はアプリの機能提供のために使用され、開発者または第三者に送信されることはありません。

4. 第三者提供・外部送信

傘おかんは、利用者の個人情報、位置情報、端末内のログを外部サーバーへ送信しません。 また、第三者に提供しません。

5. 広告・解析・トラッキング

傘おかんは、広告配信、アクセス解析、利用者を追跡するためのトラッキングを行いません。

6. 権限の管理

位置情報および通知の許可は、iOSの設定アプリからいつでも変更できます。 権限を無効にした場合、滞在場所の検知や通知機能の一部または全部が利用できなくなる場合があります。

7. ポリシーの変更

本プライバシーポリシーは、必要に応じて変更される場合があります。 重要な変更がある場合は、アプリまたは本ページ上でお知らせします。

8. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、App Storeの開発者情報に記載された連絡先、または GitHub Issues までお願いします。
